Instalar Tailwind CSS en Angular (v3.4.17)

Paso 1: Crear tu Proyecto Angular

Crea un nuevo proyecto Angular si no tienes uno:

ng new my-project
cd my-project

Paso 2: Instalar Tailwind CSS

Instala Tailwind CSS y genera el archivo de configuración:

npm install -D tailwindcss@3 postcss autoprefixer
npx tailwindcss init

Paso 3: Configurar las Rutas de las Plantillas

Añade las rutas de tus plantillas en tailwind.config.js:

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    "./src/**/*.{html,ts}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

Paso 4: Añadir Directivas Tailwind a tu CSS

Añade las directivas @tailwind a ./src/styles.css:

@tailwind base;
@tailwind components;
@tailwind utilities;

Paso 5: Iniciar el Proceso de Construcción

Ejecuta el proceso de construcción con ng serve:

ng serve

Paso 6: Usar Tailwind en tu Proyecto

Comienza a usar las clases de utilidad de Tailwind:

<h1 class="text-3xl font-bold underline">
  Hello world!
</h1>

Resumen

  • Crea un proyecto Angular.
  • Instala Tailwind CSS y genera el archivo de configuración.
  • Configura las rutas de las plantillas.
  • Añade las directivas de Tailwind a tu CSS.
  • Inicia el proceso de construcción con ng serve.
  • Comienza a usar Tailwind en tu proyecto.
