Tutorial: Cambiar Versiones de Angular CLI

Paso 1: Desinstalar la Versión Actual

Desinstala la versión global de Angular CLI:

npm uninstall -g @angular/cli

Paso 2: Limpiar la Caché de npm

Limpia la caché para evitar conflictos:

npm cache clean --force
