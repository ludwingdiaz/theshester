Usando NVM para Node.js

¿Qué es NVM?

NVM (Node Version Manager) es una herramienta que te permite instalar y cambiar entre diferentes versiones de Node.js en tu sistema.

Paso 1: Descargar NVM

Descarga NVM desde su repositorio oficial en GitHub:

https://github.com/nvm-sh/nvm

Sigue las instrucciones de instalación para tu sistema operativo.

Paso 2: Verificar la Instalación

Abre tu terminal y ejecuta el siguiente comando para verificar que NVM se instaló correctamente:

nvm --version

Deberías ver la versión de NVM instalada.

Paso 3: Listar Versiones de Node.js

Para ver las versiones de Node.js disponibles para instalar, usa:

nvm ls-remote

Para listar las versiones de Node.js que ya tienes instaladas, usa:

nvm list

Paso 4: Instalar una Versión de Node.js

Para instalar una versión específica de Node.js, usa:

nvm install 18.17.1

Reemplaza "18.17.1" con la versión que quieras instalar.

Paso 5: Usar una Versión de Node.js

Para cambiar a una versión específica de Node.js, usa:

nvm use 18.17.1

Reemplaza "18.17.1" con la versión que quieras usar.

Paso 6: Establecer una Versión Predeterminada

Para establecer una versión predeterminada de Node.js, usa:

nvm alias default 18.17.1

Reemplaza "18.17.1" con la versión que quieras usar como predeterminada.

Resumen de Comandos

  • nvm --version: Verifica la versión de NVM.
  • nvm ls-remote: Lista las versiones disponibles de Node.js.
  • nvm list: Lista las versiones instaladas de Node.js.
  • nvm install [versión]: Instala una versión de Node.js.
  • nvm use [versión]: Usa una versión de Node.js.
  • nvm alias default [versión]: Establece una versión predeterminada de Node.js.
